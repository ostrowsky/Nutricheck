## 🛠️ Краткое описание решений

### Задача 1. Интерактивный веб-дашборд (Vanilla HTML/CSS/JS)
- **Уникальная визуализация**: Реализовали радарную паутину покрытия витаминов на чистом **Canvas API** (без Chart.js или D3), что показывает глубокое понимание низкоуровневого рендеринга и тригонометрии.
- **Интеграция с API**: Дебаунсированный поиск (350мс) по базе Open Food Facts.
- **Импорт файлов**:FileReader API считывает экспорты из Google Таблиц (CSV) или отчеты Python-скрипта (JSON), мгновенно перерисовывая интерфейс.
- **Тесты**: Свой собственный микро-фреймворк автотестов (`test-runner.html`), работающий прямо в окне браузера.

### Задача 2. Python-анализатор + Docker
- **Middle+ практики**: Внедрено кэширование повторных запросов в памяти (in-memory caching), таймауты сетевых запросов и 3-ступенчатый ретрай (retry logic) с экспоненциальной задержкой в случае сбоев серверов API.
- **Кроссплатформенность**: Программное переключение потока вывода в UTF-8 для исключения ошибок Unicode на Windows.
- **Автотесты**: 100% покрытие бизнес-логики через `unittest` (тесты парсинга CSV, RDA формул, пересчета веса и фильтрации дефицитов).

### Задача 3. Apps Script в Google Таблице
- **Пакетная оптимизация**: Скрипт считывает и записывает данные одной операцией (`getValues` и `setValues`), что работает в десятки раз быстрее обычных Apps Script решений и решает проблему лимитов Google.
- **Защита данных**: Скрипт игнорирует уже обогащенные строки, предотвращая лишний сетевой трафик.
- **Логирование**: Сбои выносятся на отдельный лист «Лог», не прерывая выполнение скрипта.

## 🧑‍💻 Подход к использованию ИИ

Разработка велась строго в рамках методологии **Spec-First Development**:
1. Сначала мы зафиксировали ожидаемое поведение, форматы обмена данными (JSON-схему между Python и Web) и инварианты в папке `docs/specs/features/`.
2. Код генерировался ИИ-ассистентами по точечным спецификациям с четкими границами (что полностью исключило дрейф кода и ошибки).
3. Все сомнительные места покрывались автоматическими тестами перед финализацией.
