# Задача 2: Python-анализатор витаминов + Docker

Консольное (CLI) приложение на Python для глубокого анализа рационов питания. Оно принимает CSV-дневники питания, автоматически обогащает пищевую ценность продуктов через API Open Food Facts, вычисляет покрытие индивидуальных норм RDA и генерирует структурированные отчеты.

---

## 🚀 Команды для сборки и запуска в Docker

Контейнеризация позволяет запускать анализатор одной командой в любой операционной системе:

1. **Сборка Docker-образа**:
   ```bash
   docker build -t nutricheck-analyzer .
   ```
2. **Запуск анализа на демонстрационных данных**:
   ```bash
   docker run --rm nutricheck-analyzer --gender male --age 32 --activity moderate
   ```
3. **Запуск анализа со своим файлом дневника** (монтируем локальную папку с CSV):
   ```bash
   docker run --rm -v $(pwd)/data:/app/data nutricheck-analyzer --input data/my_diary.csv --gender male --age 32
   ```

---

## 💻 Локальный запуск без Docker

Если вы хотите запустить проект напрямую в Python-окружении:

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Запустите тесты (100% покрытие логики):
   ```bash
   python -m unittest tests/test_analyzer.py
   ```
3. Запустите анализатор:
   ```bash
   python main.py --input data/sample_food_log.csv --gender male --age 32
   ```

---

## 📊 Результаты работы и экспортируемые файлы

При каждом запуске анализатор создает следующие файлы в папке `output/`:
- `output/report.csv` — Таблица покрытия нутриентов, удобная для импорта в Excel.
- `output/report.json` — Структурированный JSON-отчёт, который можно загрузить в веб-дашборд Задачи 1 (кнопка «Импорт JSON»), чтобы мгновенно увидеть графики.
- `output/example_run.txt` — Сохраненная копия красивого консольного вывода с ASCII-шкалами.
