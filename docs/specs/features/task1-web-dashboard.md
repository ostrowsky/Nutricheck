# Спецификация: NutriCheck Web Dashboard (Task 1)

## Цель продукта
Интерактивный веб-дашборд для визуализации суточного покрытия витаминов и минералов на основе съеденных продуктов, расчета дефицитов и выдачи пищевых рекомендаций.

## Пользовательские сценарии (User-visible behavior)
1. **Настройка профиля**: Пользователь может указать свой пол (мужской/женский), возраст и уровень активности. Это пересчитывает нормы RDA (Recommended Dietary Allowance) на лету.
2. **Поиск продуктов**: Пользователь вводит название в поисковую строку. Система делает дебаунс 350мс и ищет в Open Food Facts API. Результаты отображаются в выпадающем списке.
3. **Список продуктов**: При клике на результат продукт добавляется в список с весом 100г. Пользователь может менять вес порции или удалять продукты.
4. **Визуализация (Радарная диаграмма)**: Canvas-диаграмма отображает покрытие для 8 осей (витамины A, C, D, E, B6, B12, кальций, железо). Полигон анимируется при пересчете.
5. **Детализация (Прогресс-бары)**: Все нутриенты группируются по категориям (КБЖУ, Витамины, Минералы). Каждый бар имеет цветной статус:
   - Зеленый (>=80% нормы)
   - Желтый (50-80% нормы)
   - Красный (<50% нормы, с анимацией пульсации)
6. **Рекомендации**: Выводятся карточки дефицитных элементов (покрытие < 80%) с описанием их биологической роли и списком рекомендуемых продуктов.

## Инварианты данных (Invariants)
- Суммарное потребление не может быть отрицательным.
- Качество диеты (Diet Score) рассчитывается как среднее арифметическое процента покрытия незаменимых микронутриентов (витамины и минералы, исключая макросы КБЖУ и натрий). Процент покрытия для каждого элемента ограничивается 100% сверху при расчёте общего балла диеты.
- Изменения в весе порции или настройках профиля должны приводить к мгновенному пересчёту без перезагрузки страницы.

## Граничные случаи (Edge Cases)
- **Отсутствие интернета**: Если API недоступно, выводится toast-сообщение об ошибке.
- **Некорректный импорт**: При загрузке битого CSV/JSON файлов выводится сообщение об ошибке, состояние приложения не ломается.
